Exploración de datos

En este proyecto tomé la data de una de las cosas que me apasiona como lo es el fútbol. Usé los datos de las principales ligas de Europa desde el 2008 al 2016 para poder hacer una demostración de Exploración y análisis de datos y finalmente terminar con una regresión logística. Para este proyecto usé el paquete Rstudio y su librería de exploración de datos.
