Árboles de decisión y Random Forest

En este proyecto usamos dos herramientas útiles (árboles de decisión y random forest) para determinar cuales variables inciden en las aves para que tengan mejor resolución de problemas teniendo como hipotésis que estas aves tenían mayor complejidad y recursos fonéticos. Hay una gran cantidad de variables en este estudio y no se realizó el análisis a profundidad, sino se tomó en cuenta aquellas variables que tenían mayor correlación para poder hacer una mejor visualización del árbol de decisión, que es el fín de este proyecto. Dejo abierto el análisis para discusión, ya que como resultado final podemos establecer una regresión lineal para contrastar los resultados obtenidos. La idea de este proyecto es ver el uso de la herramienta y que tan útil nos puede servir para la toma de decisiones. El proyecto esta hecho en R studio.
